<NOTAS>
En la pantalla principal aparecerán las salas de ejercicios a las que el usuario está apuntado.
Backend en Java con Springboot.
Base de datos con Firebase (Authentication, Firestore, Realtime Database).
La elaboración de una rutina consistirá en una tabla con un componente select que permitirá elegir el ejercicio y un número de repeticiones. Estructura:
	CUADRADO CON DIBUJO DEL EJERCICIO	SELECT		BOTÓN MENOS	BOTÓN MÁS	BOTÓN ELIMINAR EJERCICIO

	Solo distancias
	Ambos
